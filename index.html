How to import into QBO:
  1. Download the IIF file using the button below.
  2. In QBO, go to Settings (gear icon) → Import Data → QuickBooks Desktop (IIF).
  3. Click Browse, select the downloaded .iif file, then Import.
  4. QBO will show a summary of transactions imported. Verify the count matches the row count below.
  5. Go to Banking → For Review to confirm each transaction is categorized correctly.
Note: If QBO shows an IIF error on import, check that no account names contain special characters. The IIF format requires account names to match exactly as they appear in your QBO Chart of Accounts.
